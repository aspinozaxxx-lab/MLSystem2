# Правила Для Codex

Codex обязан сохранять минимальную архитектуру.

- Не добавлять инфраструктуру.
- Не расширять публичный интерфейс без обновления документации и тестов.
- Не добавлять функции «на всякий случай».
- Не импортировать `mlflow` вне `mlflow_adapter`.
- Не импортировать `torch` вне `train`, `models` и `inference`, кроме контрактных Protocol-типов.
- Не добавлять ФастАПИ или Ювикорн.
- Не добавлять Докер или Ансибл.
- Не добавлять Рэббит-эм-кью или Тритон.
