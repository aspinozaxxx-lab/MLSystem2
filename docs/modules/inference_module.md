# Модуль inference

## Назначение

`inference` выполняет прямой Python/PyTorch инференс без сервисной инфраструктуры. Модуль
загружает модель или чекпойнт через публичные API моделей, принимает локальную папку снимков,
выполняет инференс по тайлам или окнам и возвращает `InferenceResult`.
