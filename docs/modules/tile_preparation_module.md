# Модуль tile_preparation

## Назначение

`tile_preparation` превращает готовый `DatasetManifest` в ограниченные источники train/val батчей
тайлов. Модуль использует границы сцен для поиска соседних снимков, не заполняет соседние зоны
черным при наличии соседнего снимка, заранее готовит тайлы, держит ограниченную очередь и по умолчанию
использует 16 рабочих потоков.
