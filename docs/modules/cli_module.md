# Модуль cli

## Назначение

`cli` содержит только точки входа командной строки.

## Публичный интерфейс

Скрипты командной строки:

- `mlsystem2-train`
- `mlsystem2-infer`

Параметры:

- `--config`: путь к YAML-конфигу.
- `--run-name`: необязательное имя запуска.

## Контракты

Файла `contracts.py` нет. Точки входа CLI используют публичные контракты конвейера.

## Выходные артефакты

Прямых артефактов нет. Артефакты создают и логируют модули конвейеров.

## Что модуль НЕ делает

Не реализует бизнес-логику, не пишет в MLflow напрямую, не читает датасет напрямую, не содержит
цикл обучения и не содержит алгоритм инференса.

## Разрешенные зависимости

argparse и публичные API/contracts конвейеров.

## Запрещенные пересечения

Не импортирует приватные файлы любых модулей.

## MLflow

Напрямую не участвует.

## Временное профилирование

Время не замеряет.
